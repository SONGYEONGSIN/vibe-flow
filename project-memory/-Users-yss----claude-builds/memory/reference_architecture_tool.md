아키텍처 다이어그램(docs/architecture.png)은 **Nano Banana Pro**로 생성한다.

- API 키 위치: `~/.claude/env.local` → `NANO_BANANA_API_KEY`
- 사용 시점: 유저가 아키텍처 재생성을 요청할 때 키를 읽어서 사용
