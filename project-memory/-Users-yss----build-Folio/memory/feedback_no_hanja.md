Folio 시스템의 모든 시각 표기에 한자를 사용하지 않는다.

**Why**: 사용자가 brainstorm 단계에서 한자 인장(運經, 印章, 日 액센트) mockup을 보고 명시 거부 — "해당 시스템에는 한자 관련해서 적용하면 안되". 한국 사내 운영 시스템이지만 한자 친화 사용자가 아님.

**How to apply**:
- 로고/워드마크/명칭/액센트 마크/플레이스홀더 — 어느 곳에도 한자 글리프(漢, 運, 日, 印 등) 사용 금지
- 한자 인장/낙관 비주얼 메타포 자체를 회피 (한국 디자인 = 한자라는 가정 금지)
- 한국적 톤은 한글 워드마크 + 한글 액센트 + 토큰 색상(washi/vermilion 등)으로 표현
- 한자어의 한글 표기(예: "관제", "운영", "지휘소")는 한국어이므로 OK
- 향후 brainstorm 4-options 생성 시 한자 옵션은 사전에 제외하고 후보 구성
