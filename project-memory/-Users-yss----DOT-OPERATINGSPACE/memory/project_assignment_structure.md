### 03. 대학원 시트
학교명(B열) → 운영자(H열)

### 04. PIMS 시트
학교명(D열) → 전체사용 운영자(G열), 선택사용 운영자(I열)

### 06. 성적산출 시트
학교명(B열) → 운영자(E열)

### 07. 상담앱 시트
학교명(B열) → 운영자(F열)

### 05. 모의논술 — 사용 안 함

### 기초자료 1,2
- 서비스ID 기준 대학별 연간 건수 추출 가능
- 운영자별 연간 서비스 건수 참고

**Why:** 대학배정 UI/UX에서 배정 유형이 12+4개 = 16가지이며, 대분류에 따라 배정 가능한 유형이 다름
**How to apply:** 위자드에서 배정 유형 선택 시 대분류 조건을 반영해야 하고, 대시보드에서 유형별 진행률을 정확하게 표시해야 함
