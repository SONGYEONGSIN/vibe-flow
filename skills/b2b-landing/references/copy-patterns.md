# B2B SaaS 한국어 카피 패턴

B2B SaaS 랜딩 페이지를 위한 한국어 카피라이팅 공식과 패턴.

---

## 1. 헤드라인 공식

### 공식 A: 결과 + 시간 절약
```
[시간/비용] 을 [절약/돌려받으세요]
```
- "매주 8시간을 돌려받으세요"
- "보고서 작성에 쓰던 3시간, 이제 0분"
- "팀 미팅 시간 40%를 절약하세요"

### 공식 B: 고통 → 해방
```
[고통스러운 작업], 이제 [해방]
```
- "수작업 보고서, 이제 끝내세요"
- "엑셀 지옥에서 벗어나세요"
- "밤샘 배포, 이제 안녕"

### 공식 C: 대상 + 결과
```
[타겟 고객]을 위한 [결과]
```
- "성장하는 팀을 위한 자동화 플랫폼"
- "데이터 중심 의사결정을 원하는 리더를 위해"
- "50인 이상 팀의 협업을 바꿉니다"

### 공식 D: 수치 강조
```
[수치]로 [결과]
```
- "3단계로 팀 생산성을 2배로"
- "도입 첫 달, 업무 시간 40% 감소"
- "1,200개 팀이 선택한 이유"

### 금지 헤드라인 패턴
- "혁신적인 솔루션" → 구체적 결과로 대체
- "강력한 플랫폼" → 어떻게 강력한지 수치로
- "차세대 ..." → 현재 고객의 문제 중심으로
- "AI 기반 ..." → AI는 수단, 결과를 강조
- "원스톱 솔루션" → 무엇을 해결하는지 명시

## 2. 서브헤딩 패턴

### 공식: 구체적 방법 + 결과
```
[서비스명]이 [방법]으로 [결과]를 만들어 드립니다.
```
- "TeamFlow가 팀의 보고서를 자동으로 취합하고, 실시간 대시보드로 보여드립니다."
- "DataPipe가 흩어진 데이터를 하나로 모으고, 누구나 쉽게 분석할 수 있게 합니다."

### 규칙
- 1~2문장, 50자 이내
- 격식체 (합니다)
- 서비스명 포함
- 구체적 기능 1~2개 언급

## 3. CTA 문구 패턴

### 주요 CTA (버튼)
```
[혜택]으로 [행동]하기
```
| 상황 | CTA 문구 |
|------|----------|
| 무료 체험 | "무료로 시작하기" |
| 데모 신청 | "데모 신청하기" |
| 상담 신청 | "무료 상담 받기" |
| 자료 다운로드 | "가이드 받아보기" |
| 즉시 시작 | "지금 바로 시작하기" |

### 안심 문구 (CTA 아래)
| 상황 | 안심 문구 |
|------|-----------|
| 무료 체험 | "카드 등록 없이 14일 무료 체험" |
| 데모 | "30분 안에 핵심 기능을 확인하세요" |
| 가입 | "3분이면 충분합니다" |
| 일반 | "약정 없이, 언제든 해지 가능" |
| 보안 | "데이터는 안전하게 암호화됩니다" |

### CTA 규칙
- "하기" 종결 (비격식체): "시작하기", "신청하기", "받아보기"
- 버튼 텍스트는 7자 이내 권장
- 안심 문구는 CTA 바로 아래 배치
- 한 화면에 CTA 1개만 (시선 분산 방지)

## 4. Pain Point 카피 패턴

### 공식: 상황 묘사 + 비용
```
[구체적 상황] — [그로 인한 비용/손실]
```

### Before/After 예시

**Before (나쁜 예):**
```
데이터 관리가 어렵습니다.
```

**After (좋은 예):**
```
매주 금요일, 팀원 5명의 보고서를 수작업으로 취합하고 계신가요?
평균 3시간을 보고서 정리에 쓰고 있다면, 연간 150시간을 낭비하고 있는 겁니다.
```

### Pain Point 작성 규칙
- 의사결정자(CTO, PM, 대표) 관점
- "~하고 계신가요?" 질문형으로 공감 유도
- 비용을 시간/금액으로 정량화
- 감정적 단어: "낭비", "반복", "놓치고", "수작업"
- 3개의 Pain Point는 서로 다른 각도 (시간, 비용, 리스크)

## 5. Feature → Benefit 변환

### 변환 공식
```
[기능] → [고객이 얻는 결과]
```

| 기능 (사용 금지) | 혜택 (사용) |
|------------------|-------------|
| 실시간 동기화 | 팀 전체가 항상 같은 정보를 봅니다 |
| 자동 리포트 | 보고서 작성에 쓰던 시간을 돌려받으세요 |
| 대시보드 | 한 화면에서 모든 지표를 확인하세요 |
| API 연동 | 기존 도구와 자연스럽게 연결됩니다 |
| 권한 관리 | 필요한 사람만 필요한 정보에 접근합니다 |
| 알림 시스템 | 중요한 변경사항을 절대 놓치지 않습니다 |
| SSO 지원 | 팀원 온보딩이 3분이면 끝납니다 |

### 규칙
- "기능이 있습니다" 대신 "~할 수 있습니다", "~를 해결합니다"
- 수치가 있으면 반드시 포함
- Problem 섹션과 1:1 매핑

## 6. Social Proof 카피 패턴

### 수치형 헤딩
```
[수치]개 팀이 [동사]했습니다
```
- "1,200개 팀이 선택했습니다"
- "50,000명이 매일 사용합니다"
- "국내 100대 기업 중 23개사가 도입했습니다"

### 신뢰 배지 문구
| 유형 | 문구 |
|------|------|
| 운영 기간 | "2019년부터 서비스" |
| 평점 | "4.8점 (리뷰 2,300개)" |
| 인증 | "ISO 27001 인증" |
| 수상 | "2024 서비스 대상" |
| 언론 | "매경, 조선일보 소개" |

## 7. FAQ 카피 패턴

### B2B 필수 질문 5가지

```
Q: 가격은 어떻게 되나요?
A: 팀 규모와 사용량에 따라 맞춤 견적을 제공합니다. 무료 상담을 통해 정확한 가격을 안내받으실 수 있습니다.

Q: 기존에 사용하는 도구와 연동이 되나요?
A: Slack, Notion, Jira 등 주요 협업 도구와 기본 연동을 지원합니다. 추가 연동이 필요하시면 API를 통해 가능합니다.

Q: 데이터 보안은 어떻게 관리되나요?
A: 모든 데이터는 AES-256 암호화로 저장되며, ISO 27001 인증을 받은 인프라에서 운영됩니다.

Q: 도입까지 얼마나 걸리나요?
A: 기본 설정은 30분이면 완료됩니다. 전담 매니저가 온보딩을 지원해 드립니다.

Q: 해지는 어떻게 하나요?
A: 약정이 없으며, 설정에서 언제든 해지 가능합니다. 해지 후에도 30일간 데이터를 보관합니다.
```

### FAQ 규칙
- 질문: "~인가요?", "~되나요?" (비격식체)
- 답변: "~합니다", "~됩니다" (격식체)
- 답변 끝에 행동 유도 가능 ("무료 상담을 통해...")
- 3~5개 (너무 많으면 압도, 너무 적으면 불안)


## 9. 격식체/비격식체 가이드

| 위치 | 어조 | 예시 |
|------|------|------|
| 히어로 헤드라인 | 비격식체 (해요/하세요) | "매주 8시간을 돌려받으세요" |
| 서브헤딩 | 격식체 (합니다) | "자동으로 보고서를 생성합니다" |
| CTA 버튼 | 비격식체 (하기) | "무료로 시작하기" |
| Pain Point | 비격식체 (계신가요?) | "수작업으로 취합하고 계신가요?" |
| Solution | 격식체 (합니다) | "팀 생산성을 높여드립니다" |
| FAQ 질문 | 비격식체 (인가요?) | "해지는 어떻게 하나요?" |
| FAQ 답변 | 격식체 (합니다) | "언제든 해지 가능합니다" |
| 안심 문구 | 격식체 (합니다/없이) | "카드 등록 없이 시작" |

### 핵심 원칙
- 친근함이 필요한 곳: 비격식체 (헤드라인, CTA, 질문)
- 신뢰감이 필요한 곳: 격식체 (설명, 답변, 본문)
- 한 섹션 내에서는 일관된 어조 유지
